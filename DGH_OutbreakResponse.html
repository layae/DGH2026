🔗 접촉자 추적 (Contact Tracing)

원칙: 노출원 확인 → 접촉자 목록 작성 → 추적·모니터링 → 격리/치료

인도주의 현장 한계:
· 인구 이동 잦음 → 접촉자 소재 불명
· 신뢰 부족 → 숨김·도주
· 자원 부족 → 대규모 추적 불가

에볼라 링 전략:
확진자 → 모든 접촉자(1링) → 접촉자의 접촉자(2링) 모두 백신접종·모니터링

COVID-19 교훈 적용:
디지털 도구·커뮤니티 자원봉사자 활용 → 자원 한계 극복

📊 인도주의 현장 역학 조사 특수 고려

  • 분모 불명확: 캠프 등록 인구 vs 실제 인구 차이 → 발병률 부정확
  • 이동 인구: 코호트 유지 불가 → 단면 조사 반복
  • 증례 정의 단순화: 실험실 불가 시 임상 기준만으로 운영
  • Lot Quality Assurance Sampling (LQAS): 소규모 표본으로 신속 접종률 평가
  • Mortality survey: CMR 추정; 두 단계 군집 표본 (30×30)
  • SMART survey: 영양 상태 신속 평가 (W/H·MUAC·부종)

30×30 클러스터 방법: 30개 지역·30명 무작위 표본 → 접종률 추정. 신속하고 현장 적용 가능.

⚠️ 시험 함정 4개
영양불량 소아 홍역 = 경증?
영양불량(특히 비타민A 결핍) 소아에서 홍역 CFR 20~30%까지 상승 (선진국 0.1%와 비교). 각막 손상으로 실명 위험. 인도주의 현장에서 홍역 확진 즉시 비타민A 투여가 표준 치료. 영양 상태가 감염병 CFR을 결정적으로 높임.
에볼라 = 비말 전파?
에볼라는 직접 접촉(체액)·안전하지 않은 매장 전파. 비말 전파 아님. 샘플 Q7에서 확인 — IPC는 증상 발현 즉시(증상 전 전파력 낮음) 시작. 비말 격리가 아닌 접촉 격리가 핵심. 안전 매장 = 인도주의 현장 에볼라 통제 핵심 요소.
링 백신 = 확진자만 접종?
링 백신 전략은 확진자의 모든 접촉자(1링) + 그 접촉자들의 접촉자(2링)에게 접종. 에볼라 rVSV-ZEBOV는 노출 후에도 효과 (노출 후 예방 가능). 확진자 본인은 이미 감염 상태이므로 접종 대상 아님.
콜레라 OCV = 1회 접종으로 완전 면역?
Shanchol은 기본 2회 접종 (2주 간격). 1회 접종도 단기(6개월) 보호 효과 있어 아웃브레이크 초기 대규모 1회 캠페인 사용. 완전 면역(2년)은 2회 접종 필요. 자원 제한 시 1회 전략 허용 but "1회로 완전 면역" = 오답.

샘플 Q7 에볼라 IPC 연계: 정답(B) — 증상 발현 즉시(vague undifferentiating symptoms 단계부터) IPC 조치. 출혈이 나타날 때까지 기다리면 이미 전파 발생. 에볼라 IPC의 핵심은 조기 격리 + 체액 접촉 차단. 안전 매장이 지역사회 전파 차단에 결정적. 옵션 C("출혈 시 조치")는 전형적 오답 함정.
