🎯 Eschar (가피) = 시험 핵심 단서: 털진드기/진드기 교상 부위 → 괴사성 가피 형성. Scrub typhus(50–80%)와 African tick bite fever(다발성)에서 특징적. 겨드랑이·서혜부·귀 뒤 등 은닉 부위 검색 필수. Eschar 발견 = 즉시 Doxycycline 시작.
병태생리 — 혈관내피 침범
  • 리케차 = 절대세포내 기생 → 혈관내피세포 침범
  • 혈관염 → 모세혈관 누출 → 발진·부종·저혈압
  • 심한 경우: 폐부종·뇌부종·급성 신부전·DIC
  • Epidemic typhus: 인이(body louse) 배설물 → 피부찰과상 통해 감염
  • Scrub typhus: Orientia (구 Rickettsia)로 분류 변경 — 유전적 다양성 높음
  • Q fever 만성화: 심내막염(인공판막·심장병 기저) — 1–2년 치료 필요
진단 — 현장 접근 전략
  • 임상 진단 + 역학 = 치료 시작 기준 (검사 결과 기다리지 않음)
  • Weil-Felix test: 저렴하나 민감도·특이도 낮음 — LMIC 선별에만 사용
  • IFA (간접형광항체): 표준 혈청검사 — 급성기·회복기 쌍혈청
  • PCR (혈액·eschar 조직): 가장 신속·정확, 항생제 전 시행
  • Eschar 생검: PCR + 조직학 — Scrub typhus 확진에 유용
  • 혈액검사: 혈소판감소·간수치 상승·저나트륨혈증 → 리케차 시사

Epidemic Typhus — 인도주의 위기 맥락: 체이(body louse) 만연 환경 = 과밀 난민캠프·전쟁·기근 상황. Brill–Zinsser disease: 과거 감염 후 수십 년 뒤 재활성화(스트레스·면역저하). 대유행 예방 = DDT·퍼메트린 살충제 + 의류 소독. 르완다 대학살 난민캠프에서 집단발생(1997) 교훈.
⚠ 시험 함정 4
🔴
RMSF 발진 — 손발바닥 포함 여부가 왜 중요?
RMSF는 손발바닥에 발진이 생기는 드문 질환. 초기에는 체간 시작 후 원심성으로 퍼짐 → 손발바닥 도달. 점상출혈·자반으로 진행 시 중증 신호. 뇌막구균혈증과 감별 필요.
🌿
Scrub typhus — 아시아 여행자 발열 감별?
아시아·태평양 여행 후 발열+eschar → Scrub typhus 1순위. Eschar 없어도 배제 불가(50%는 없음). 말라리아 음성 + 혈소판감소 + eschar 흔적 → Doxy 경험적 치료 시작.
👶
소아 RMSF에서 Doxycycline 사용 가능한가?
Yes, 8세 미만도 사용 가능. RMSF는 치료 지연 시 CFR 20–25% → 뼈 착색 위험보다 생명이 우선. CDC·WHO 모두 소아 리케차에서 Doxy 권고. Chloramphenicol은 회색아증후군 위험으로 비선호.
❤️
Q fever 완치 후 추적이 필요한가?
Yes. 급성 Q fever 환자 중 심장판막 이상·임신부·면역저하 → 만성화(심내막염) 고위험. 항체 역가 추적 (Phase I IgG >1:800 = 만성 기준). 고위험군 예방 연장 치료 검토.
