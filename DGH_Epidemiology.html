🔬 연구 설계 위계

  • RCT — 인과관계 최강; LMIC 시행 어려움
  • 코호트 — Incidence·RR 산출; 전향/후향
  • 환자-대조군 — 희귀질환 적합; OR 산출
  • 단면 연구 — 유병률만; 인과관계 불가
  • 생태 연구 — 집단 수준; ecological fallacy ⚠
  • 체계적 리뷰/메타분석 — 증거 피라미드 최상위

▶ LMIC 현실: 후향 코호트·단면 연구 다수 → 인과추론 한계

📐 핵심 측정 지표

  • Incidence rate = 신규 발생 / person-time
  • Attack rate = 노출 후 발병률 (아웃브레이크)
  • CFR = 사망/확진; 분모 정의가 핵심
  • RR = 코호트; RR=1 → 연관 없음
  • OR = 환자대조군; 희귀질환 시 RR 근사
  • AR% = (RR-1)/RR × 100 → 예방가능분율
  • NNT = 1/ARR → 낮을수록 효과적 중재

🔍 선별검사 원칙 (Wilson-Jungner)

  • 중요한 건강 문제일 것
  • 수용 가능한 치료법 존재
  • 잠복기/초기에 발견 가능
  • 적합한 검사 도구 존재 (sensitivity 우선)
  • 비용-효과적이어야 함

Sensitivity = TP/(TP+FN) → rule OUT (SNOUT)
Specificity = TN/(TN+FP) → rule IN (SPIN)
PPV/NPV는 유병률 따라 변동 → LMIC 고유병률 → PPV ↑

📈 ROC 커브 & 역학 적용

  • AUC: 0.5=무작위, 1.0=완벽
  • Cut-off ↓ → sensitivity ↑, specificity ↓
  • 대규모 선별: sensitivity 우선 (HIV ELISA)
  • 확진: specificity 우선 (HIV Western blot)

LR(+) = Sens/(1-Spec)
LR(+) >10 → 질환 있을 가능성 크게 ↑
LR(-) <0.1 → 질환 배제에 유용

🌍 LMIC 역학 특수 고려사항

  • 감시 체계 취약: under-reporting → 실제 발병률 과소 추정
  • 사망 원인 불명확: verbal autopsy 사용 → 정확도 한계
  • 이중 부담: 감염병 + NCD 동시 → 복합 역학
  • 연령 구조: 피라미드형 → 소아 사망이 DALYs 주도
  • DALY = YLL + YLD; GBD 연구 핵심 지표
  • QALY: $50,000/QALY 이하 → 비용효과적
  • HALE: 단순 기대수명보다 정책 관련성 높음
  • Bradford Hill 기준: 강도·일관성·특이성·시간성·생물학적 타당성

⚠️ 시험 함정 (Exam Traps)

🪤 OR ≠ RR (흔한 질환) 환자-대조군 OR은 희귀질환에서만 RR 근사. 유병률 >10%이면 OR이 RR 크게 과대추정. "환자-대조군에서 RR 직접 계산" → 오답.
🪤 PPV는 유병률 의존 sensitivity/specificity는 검사 고유 특성. 저유병률 집단(일반인 HIV 선별)에서는 PPV 낮아 FP 많음. 고유병률 LMIC → PPV 상승 효과 기억.
🪤 단면연구 → 인과관계 불가 단면연구는 prevalence만 측정. "단면연구에서 인과관계 증명" → 항상 오답. 만성질환은 생존 편향으로 prevalence 과대; incidence와 구분 필수.
🪤 Confounding ≠ Effect modification 교란변수는 통계 조정으로 제거 가능. 효과변경인자(effect modifier)는 층화 분리 보고 필요 — 조정하면 정보 손실. 예: 침상망 효과가 연령군별 다름 → interaction.
